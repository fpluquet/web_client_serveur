Introduction

Cette application démontre la différence entre un client léger et un client lourd dans l'architecture web.

  • Client Léger (Thin Client) - Pages générées côté serveur, rechargement complet à chaque action
  • Client Lourd (Rich Client) - Application JavaScript qui communique avec une API, mise à jour dynamique sans rechargement

Client Lourd (Rich Client)

Cette interface utilise JavaScript pour communiquer avec une API REST et mettre à jour la page dynamiquement.
